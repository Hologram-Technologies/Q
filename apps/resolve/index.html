Holo Resolve

One input for every name the internet has — κ labels, IPFS CIDs, Ethereum hashes, SRI, onion, truenames. Every byte re-derives before you see it.

100% serverless. This page is static; the resolver runs in your browser. Content-derived names are translated onto the κ substrate and fetched from untrusted mirrors — the first bytes that re-derive to their hash win; liars and dead mirrors are indistinguishable from silence. blake3 verification runs through the holospaces runtime (the same wasm this bundle boots on).
